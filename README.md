# **ErikNeph_Obsidian**

**ErikNeph_Obsidian** - представляет собой базу знаний написанной с помощью приложения [Obsidian](https://obsidian.md/), использующий популярный формат [markdown](https://www.markdownguide.org/).Obsidian служит как раз для упрощения работы с markdown, что заметно упростит вашу работу с markdown. Моя база знаний может послужить примером создания вашей базы знаний (напримере моей). Я старался сделать мое хранилище Obsidian максимально понятным и интуитивным, вложенность папок в моем хранилище три уровня.

**Я использую следующие сторонние плагины для Obsidian (на что и вам рекомендую данные плагины для работы в Obsidian):**

- Calendar
- Iconize
- Git
- Beautitab
- Style Settings
- Minimal Theme Settings
