## üîπ–£—Å–ª–æ–≤–Ω—ã–µ –∫–æ–Ω—Å—Ç—Ä—É–∫—Ü–∏–∏: `if`, `elif`, `else`

**–ß—Ç–æ —ç—Ç–æ —Ç–∞–∫–æ–µ:**  
–£—Å–ª–æ–≤–Ω—ã–µ –∫–æ–Ω—Å—Ç—Ä—É–∫—Ü–∏–∏ –ø–æ–∑–≤–æ–ª—è—é—Ç –≤—ã–ø–æ–ª–Ω—è—Ç—å —Ä–∞–∑–Ω—ã–µ –±–ª–æ–∫–∏ –∫–æ–¥–∞ –≤ –∑–∞–≤–∏—Å–∏–º–æ—Å—Ç–∏ –æ—Ç –∏—Å—Ç–∏–Ω–Ω–æ—Å—Ç–∏ –∑–∞–¥–∞–Ω–Ω—ã—Ö —É—Å–ª–æ–≤–∏–π.  
_–ü—Ä–∏–º–µ—Ä –∏–∑ –∂–∏–∑–Ω–∏:_  
–ï—Å–ª–∏ —Ç–µ–º–ø–µ—Ä–∞—Ç—É—Ä–∞ –Ω–∏–∂–µ 0¬∞, –Ω–∞–¥–µ–≤–∞–µ—Ç–µ –∑–∏–º–Ω—é—é –∫—É—Ä—Ç–∫—É; –µ—Å–ª–∏ –æ—Ç 0 –¥–æ 15¬∞ -- –ª—ë–≥–∫—É—é –∫—É—Ä—Ç–∫—É; –µ—Å–ª–∏ –≤—ã—à–µ -- —Ñ—É—Ç–±–æ–ª–∫—É.

**–°–∏–Ω—Ç–∞–∫—Å–∏—Å –∏ –ø—Ä–∏–º–µ—Ä:**


**–û—Å–Ω–æ–≤–Ω—ã–µ –º–æ–º–µ–Ω—Ç—ã:**

- **`if`**¬†–ø—Ä–æ–≤–µ—Ä—è–µ—Ç —É—Å–ª–æ–≤–∏–µ: –µ—Å–ª–∏ –∏—Å—Ç–∏–Ω–Ω–æ, –≤—ã–ø–æ–ª–Ω—è–µ—Ç—Å—è —Å–≤—è–∑–∞–Ω–Ω—ã–π –±–ª–æ–∫, –∞ –ø–æ—Å–ª–µ–¥—É—é—â–∏–µ –ø—Ä–æ–≤–µ—Ä–∫–∏ –∏–≥–Ω–æ—Ä–∏—Ä—É—é—Ç—Å—è.
- **`elif`**¬†–ø–æ–∑–≤–æ–ª—è–µ—Ç –∑–∞–¥–∞—Ç—å –¥–æ–ø–æ–ª–Ω–∏—Ç–µ–ª—å–Ω—ã–µ –ø—Ä–æ–≤–µ—Ä–∫–∏, –µ—Å–ª–∏ –ø—Ä–µ–¥—ã–¥—É—â–∏–µ —É—Å–ª–æ–≤–∏—è –ª–æ–∂–Ω—ã.
- **`else`**¬†–≤—ã–ø–æ–ª–Ω—è–µ—Ç—Å—è, –µ—Å–ª–∏ –Ω–∏ –æ–¥–Ω–æ –∏–∑ –ø—Ä–µ–¥—ã–¥—É—â–∏—Ö —É—Å–ª–æ–≤–∏–π –Ω–µ –∏—Å—Ç–∏–Ω–Ω–æ.

**–¢–∏–ø–∏—á–Ω—ã–µ –æ—à–∏–±–∫–∏:**

- –ü—Ä–æ–ø—É—â–µ–Ω–Ω–æ–µ –¥–≤–æ–µ—Ç–æ—á–∏–µ –ø–æ—Å–ª–µ —É—Å–ª–æ–≤–∏–π.
- –ù–µ–ø—Ä–∞–≤–∏–ª—å–Ω—ã–µ –æ—Ç—Å—Ç—É–ø—ã –≤–Ω—É—Ç—Ä–∏ –±–ª–æ–∫–æ–≤.
- –ò—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ –æ–ø–µ—Ä–∞—Ç–æ—Ä–∞ –ø—Ä–∏—Å–≤–∞–∏–≤–∞–Ω–∏—è (`=`) –≤–º–µ—Å—Ç–æ –æ–ø–µ—Ä–∞—Ç–æ—Ä–∞ —Å—Ä–∞–≤–Ω–µ–Ω–∏—è (`==`).


## üîπ–í–ª–æ–∂–µ–Ω–Ω—ã–µ —É—Å–ª–æ–≤–Ω—ã–µ –æ–ø–µ—Ä–∞—Ç–æ—Ä—ã

**–ß—Ç–æ —ç—Ç–æ —Ç–∞–∫–æ–µ:**

**–ü—Ä–∏–º–µ—Ä -- –ø—Ä–æ–≤–µ—Ä–∫–∞ —á–∏—Å–ª–∞ –∏ –µ–≥–æ —á–µ—Ç–Ω–æ—Å—Ç–∏:**

**–°–æ–≤–µ—Ç:**

## üîπ–õ–æ–≥–∏—á–µ—Å–∫–∏–µ –æ–ø–µ—Ä–∞—Ç–æ—Ä—ã: `and`, `or`, `not`

**–ù–∞–∑–Ω–∞—á–µ–Ω–∏–µ:**  
–õ–æ–≥–∏—á–µ—Å–∫–∏–µ –æ–ø–µ—Ä–∞—Ç–æ—Ä—ã –ø–æ–∑–≤–æ–ª—è—é—Ç –æ–±—ä–µ–¥–∏–Ω—è—Ç—å –Ω–µ—Å–∫–æ–ª—å–∫–æ —É—Å–ª–æ–≤–∏–π –≤ –æ–¥–Ω–æ –≤—ã—Ä–∞–∂–µ–Ω–∏–µ.

    –í–æ–∑–≤—Ä–∞—â–∞–µ—Ç `True`, –µ—Å–ª–∏ –≤—Å–µ —É—Å–ª–æ–≤–∏—è –∏—Å—Ç–∏–Ω–Ω—ã.

    
- **or**

    
- **not**  
    –ò–Ω–≤–µ—Ä—Ç–∏—Ä—É–µ—Ç –ª–æ–≥–∏—á–µ—Å–∫–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ.

## üîπ–ö–æ–Ω—Å—Ç—Ä—É–∫—Ü–∏—è `match-case` (Python 3.10+)
